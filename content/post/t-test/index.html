Introduction

A t-test is an inferential statistic used to draw conclusions based on comparisons between means. Like any statistical test, it weights the estimated effect by the precision of the estimate.

\[ \text{test statistic} = \frac{\text{estimate} âˆ’ \text{Null Hypothesized Value}}{\text{Variability}} \]

The shape this formula takes depends on the type of test we want to perform. T-tests are used to compare means, so the value of estimate and the Null-Hypothesized Value will be means and the variability will be the standard error of the mean.
